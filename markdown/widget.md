# Supported Widgets (MVP)

This project supports a **limited, well-defined set of widgets** for composing landing pages.  
Each widget represents a **semantic block of content**, not a visual variant.

The following widgets define the **initial MVP scope**.

---

## Hero Banner

- **Category:** `header`
- **Purpose:**  
  Communicate the main message of the page and guide the user toward the primary action.
- **Typical content:**  
  Headline, subheadline, primary call-to-action.
- **Notes:**  
  This widget represents the primary entry point of the page.

---

## Text Section

- **Category:** `body`
- **Purpose:**  
  Provide free-form editorial content for explanations or storytelling.
- **Typical content:**  
  Section title, rich text.
- **Notes:**  
  Used for descriptive or informational sections.

---

## Feature List

- **Category:** `body`
- **Purpose:**  
  Present a structured list of benefits or key features.
- **Typical content:**  
  Section title, list of features (text + optional icon).
- **Notes:**  
  Optimized for scannability and clarity.

---

## Call To Action (CTA Block)

- **Category:** `body`
- **Purpose:**  
  Encourage a specific user action.
- **Typical content:**  
  Short message and primary action button or link.
- **Notes:**  
  Used to drive conversions at key points in the page.

---

## Simple Footer

- **Category:** `footer`
- **Purpose:**  
  Close the page with secondary information and links.
- **Typical content:**  
  Copyright, basic navigation links, legal information.
- **Notes:**  
  Minimal by design, not intended for complex navigation.

---

## Design Principles

- Widgets represent **content types**, not visual styles.
- Visual variations are handled by theming, not by new widget types.
- Each widget has a **single, clear responsibility**.
- The widget set is intentionally limited and can be extended in future phases.

---

## Scope

This widget list defines the **initial MVP**.  
Additional widgets or variants will be introduced only when driven by clear content or business requirements.

---

## Outcome

With this widget set, it is possible to compose a complete, SEO-friendly landing page using structured data only.
